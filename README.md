# Pathfinding Visualizer

A Java Swing based Pathfinding Visualizer project.

## Features
- BFS Algorithm
- DFS Algorithm
- Dijkstra Algorithm
- Interactive Grid
- Wall Creation
- Path Animation

## Technologies Used
- Java
- Swing
- Data Structures & Algorithms

## How to Run

1. Open project in IntelliJ IDEA
2. Run Main.java
3. Visualize pathfinding algorithms
